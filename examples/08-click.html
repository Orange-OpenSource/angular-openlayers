Click example

The following example demonstrates how to add a click handler.
Click on the map to obtain console log "singleclick catched on aol.Model.Map() at lCoordinates= [XXXXX, YYYYY] , run alert on each corresponding Feature".
For each feature available at this point an alert is triggered (only one node in this sample) with following content : "singleclick catched on with satanic properties : pFeature.values_.properties=666"
"Ctrl" usage is also furnished as example with console log "ctrl + singleclick catched on aol.Model.Map()".
